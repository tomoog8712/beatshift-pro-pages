BeatShift Pro

プライバシーポリシー

最終更新日: 2026年7月23日

Studio Vivi Music（以下「当方」）は、モバイルアプリケーション「BeatShift Pro」（以下「本アプリ」）における ユーザーの情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

1. 収集する情報

本アプリでは、以下の情報が端末内に保存される場合があります。

1-1. 練習設定の保存

ユーザーが保存した練習設定（テンポ、モード、音量など）は端末内にのみ保存され、 当方のサーバーへ送信されることはありません。

1-2. アプリ内の状態

レビュー案内の表示状態など、アプリの動作に必要な最小限の設定が端末内に保存されることがあります。

2. 情報の利用目的

  • メトロノーム機能の提供
  • 練習設定の端末内保存・読み込み
  • アプリの品質向上・不具合対応

3. 第三者への提供

本アプリは広告 SDK や解析 SDK を使用していません。 当方がユーザー情報を第三者に販売・提供することはありません。

4. データの保存期間

端末内に保存される練習設定等は、アプリを削除するまで端末に残ります。

5. お子様のプライバシー

本アプリは、13 歳未満のお子様から故意に個人情報を収集することを目的としていません。

6. 本ポリシーの変更

必要に応じて本ポリシーを変更することがあります。変更後は本ページに掲載した時点で効力を生じます。 重要な変更がある場合は、アプリ内または本ページでお知らせします。

7. お問い合わせ
