GitHub Pages の URL 例

リポジトリを GitHub に push し、Settings → Pages → Source を Deploy from a branch、Branch を main / docs に設定すると、次の URL で公開されます。

  • https://<ユーザー名>.github.io/<リポジトリ名>/support.html
  • https://<ユーザー名>.github.io/<リポジトリ名>/privacy.html

App Store Connect には上記 URL をそのまま入力してください。
